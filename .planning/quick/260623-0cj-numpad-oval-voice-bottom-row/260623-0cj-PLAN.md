# Quick Task 260623-0cj — 数字小键盘重设计

## Task

重新设计记账录入页的数字小键盘（`数字小键盘` = `VoiceRecordBar` 语音键 +
`SmartKeyboard` 数字网格）：

1. **语音录入键不要顶边、改成椭圆形按键** — 现状是 38dp 满宽贴边色带，
   改成居中、不顶边的椭圆胶囊（Stadium 全圆角）。
2. **把最下面一排按键高度调低一些** — 底部动作排（删除 / ¥货币 / 保存）
   从与数字键等高，改为更矮。

先出 HTML 设计稿经用户确认，再写 Flutter 代码。

## Locked design decisions (用户确认)

- **语音键椭圆宽度 = 200dp**（A「贴合文字」≈150 与 B「内缩 20dp」≈310 的中间值，
  居中、上下留白、不顶边）。来源：`mocks/numpad-voicekey-midpoint.html` 方案 M。
- **底排高度 = 40dp（−23%）**：用户「下排40dp没有问题」。实现为
  `max(40, keyHeight × 0.77)` —— 标准/小屏恒 40dp，大屏按比例略增（保持 −23% 观感），
  永不低于 40dp。数字键 keyHeight（≥48dp 下限）不动。
- 仅改键盘区；数字键 1–9 / 00 / 0 / . 的尺寸·配色·间距·tabular 字形全不变。
- 配色沿用 `AppPalette`（joyLight / joyText / daily / FAB 樱粉渐变），零裸 hex。
