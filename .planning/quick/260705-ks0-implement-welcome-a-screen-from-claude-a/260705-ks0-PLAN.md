       `child`, `period` (Duration, default 6s), `phase` (Duration offset, default zero) so
       the three cluster elements on the welcome page can float out of phase

       `SingleTickerProviderStateMixin`; drive translateY with a sine curve

       visible restart snap.
    2. `DriftPetal` — a small petal shape: `Container` with size param, color param
       (caller passes `palette.joy` or `palette.accentPrimary`), opacity param, and

       `BorderRadius.only(topLeft: Radius.circular(size*0.62), bottomRight: Radius.circular(size*0.62))`
       inside a `Transform.rotate`. Loop the design `drift` keyframes: translateY 0→-9→0
       combined with rotation 45°→60°→45°, same controller pattern as FloatyLoop, with
       `period`/`phase` params.
