## What Was Built

### Task 1

`HomeSelectedMonth` notifier appended to `state_home.dart`:
- State type: `({int year, int month})` named record
- `keepAlive: true` — persists across tab switches
- Methods: `selectMonth(year, month)`, `prevMonth()`, `nextMonth()`
- `prevMonth`/`nextMonth` use `DateTime(state.year, state.month ± 1)` for correct month overflow (e.g. Jan-1 → Dec previous year)
- Generated `homeSelectedMonthProvider` in `state_home.g.dart`

`HeroHeader` updated:
- Added `onPrevMonth` and `onNextMonth` required `VoidCallback` parameters
- Left `IconButton(Icons.chevron_left)` + right `IconButton(Icons.chevron_right)` flank the existing month label `GestureDetector`
- Icon size 20, `textSecondary` color, zero padding, 28×28 min constraints

### Task 2

`home_screen.dart`:
- Watches `homeSelectedMonthProvider` for `year`/`month` (replaces `DateTime.now()`)
- `currentMonthStart` / `currentMonthEnd` now derived from provider state
- `todayTransactionsProvider` still uses its own internal `DateTime.now()` — today's transactions remain always-today
- `HeroHeader` receives `onPrevMonth`, `onNextMonth`, `onDateTap: () => _showMonthPicker(...)`
- `_showMonthPicker` static method: `showDialog<({int year, int month})?>` + applies result to `homeSelectedMonthProvider.notifier.selectMonth`
- `_MonthPickerDialog`: `StatefulWidget` with dialog-internal `_dialogYear`, year clamped `[2000, 2099]` (T-s07-02), 4×3 `GridView.count` of `_MonthChip` (FilledButton.tonal if selected, OutlinedButton otherwise), year nav row with left/right `IconButton`s
- view-all `GestureDetector.onTap`: `listFilterProvider.notifier.selectMonth(now.year, now.month)` + `selectedTabIndexProvider.notifier.select(1)`

ARB files (all 3): added `homeMonthPickerTitle` + `homeMonthPickerClose`.
