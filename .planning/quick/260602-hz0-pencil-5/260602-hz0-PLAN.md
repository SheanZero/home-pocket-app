# Quick Task 260602-hz0 — 首页三色圆环重设计（5 方案）

**Date:** 2026-06-02
**Type:** Design exploration (UI/data-viz)
**Goal:** 重新设计首页「三色圆环」(HappinessRingsPainter)，调研市面常见实现，产出 5 种不同的圆环设计方案，目标更有亲和度、颜色过渡更自然。

## Context

- 现状：`lib/features/home/presentation/widgets/painter/happiness_rings_painter.dart` —
  3 个**同心环**（Apple-Watch 风格），`CustomPainter` + `drawArc` + `SweepGradient`，
  `StrokeCap.round`，stroke 8 / gap 4，120×120，颜色取 ADR-018 `palette.daily/joy/shared`。
  内环用 `Color.lerp(daily, joy)`。
- 调色板：ADR-018 Teal Clarity（日常 teal `#1C7A86` / 悦己 gold `#F0A81E` / 共享 blue `#5B8AC4`）。

## Constraint discovered (deviation driver)

Pencil MCP (`mcp__pencil__*`) 在本环境**无法落盘**（已知限制，STATE.md D-03b 亦记录），
且 `gsd-executor` 子代理被剥离 MCP 工具（claude-code#13898）。
→ Pencil 既不能由 executor 执行，主代理写入也不持久、`get_screenshot` 读到的是旧盘文件。
**决策：** 交付改为自包含 HTML+SVG 文稿（可在浏览器查看 / 可入库 / 可评审），保证可验证与持久化。

## Tasks

1. 调研市面 ring/donut 实现（Apple Fitness/Card、Copilot/Emma/Monzo/Cleo），
   提炼「自然过渡」(OKLCH/类比色/接缝处理) 与「亲和度」(圆头/粉彩/暖心圆心/有机形态) 技法 → RESEARCH.md。
2. 产出 5 个差异化方向（D1 柔光三环 / D2 渐变甜甜圈 / D3 日出仪表弧 / D4 流体花瓣环 / D5 堆叠+吉祥物）。
3. 实现为 `docs/design/home-ring-redesign.html`：5 张卡片 + 浅/深色切换 + 对比表 + 推荐路径，
   全部基于 ADR-018 token，深色用 dark 变体校验。
4. 浏览器实测验证（chrome-devtools 截图，浅色 + 深色，0 console error）。

## Verify

- HTML 在浏览器渲染 5 个清晰可辨的圆环，浅/深色均正常，无 console error。
- 每个方案标注：隐喻 / 过渡技法 / 亲和度 / 信息精度 / 色觉友好 / 改造成本。
- 给出可落地的推荐路径（与现有 CustomPainter 的兼容性）。
