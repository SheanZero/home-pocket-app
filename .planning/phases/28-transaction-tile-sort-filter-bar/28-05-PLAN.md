| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-28-05-02 | Information Disclosure | Search field text in bar visible to other UI elements | accept | In-memory only; local-first app with no network; search field collapses when empty |
| T-28-05-03 | Tampering | Sort menu or ledger chip state manipulation | accept | listFilterProvider is a single-book provider parameterized by bookId; no cross-book data leakage at UI layer |
| T-28-05-SC | Tampering | npm/pip/cargo installs | accept | No new packages |

No new trust boundary — inherits existing repository/decryption guarantees. Text search is Dart-side string matching; no SQL or filesystem access.
