    ExchangeRateCacheService.getRate(DateTime date, String currency) → Future<RateResult>:

    CACHE HIT (RATE-02): `repository.findByDate(currency, date)` returns row AND row is not a correctable proxy → return `RateCached(rate, currency, cachedDate=row.rateDate, source=row.source)`. Zero API calls.

    D-03 correctable proxy: row is correctable if `row.actualRateDate != null && row.rateDate.isBefore(today) && row.fetchedAt.isBefore(today)`. If correctable → fall through to FETCH (one-time re-fetch). After re-fetch, the new upserted row has fetchedAt=now, so next access finds fetchedAt >= today → not correctable → no further re-fetch.

    Today's rate TTL (D-01): row for today (rateDate == today) with fetchedAt >= today midnight is considered valid for the day — return as RateCached. Cross-midnight invalidation happens naturally because `fetchedAt.isBefore(today)` becomes true after midnight.

    D-05 connectivity gate: before any network call, `await Connectivity().checkConnectivity()`. If result is all `ConnectivityResult.none` → skip to CACHE FALLBACK.

    D-06 in-memory cooldown: `DateTime? _cooldownUntil`. If `_cooldownUntil != null && DateTime.now().isBefore(_cooldownUntil!)` → skip to CACHE FALLBACK. Set on all-sources-fail: `_cooldownUntil = DateTime.now().add(const Duration(minutes: 1))`.

    FETCH (on cache miss or correctable proxy):
    1. Try `_apiClient.fetchRate(currency, date)`. On success: build ExchangeRate model with source from client result, upsert via `_repository.upsert(...)`, call `_pruneStaleCache()` as fire-and-forget (`unawaited` or ignore return), return `RateFetched(rate, currency, rateDate=date, actualDate=clientResult.actualRateDate, source=clientResult.source)`.
    2. On `ExchangeRateApiException` or any exception from _apiClient: set `_cooldownUntil = DateTime.now().add(const Duration(minutes: 1))` (D-06) → fall through to CACHE FALLBACK.

    CACHE FALLBACK:
    1. `_repository.findLatestNonManual(currency)` → if found → return `RateFallback(rate, currency, cachedDate=row.rateDate)` (D-07).
    2. `_repository.findLatest(currency)` (may be manual row) → if found AND source == 'manual' → return `RateManual(rate, currency, cachedDate=row.rateDate)` (D-07 lowest priority).
    3. Nothing found → return `RateUnavailable(currency)` (D-08).

    D-09 _pruneStaleCache: compute cutoff = `DateTime.utc(DateTime.now().year - 2, DateTime.now().month, DateTime.now().day)`. Call `_repository.deleteOlderThan(cutoff)`. Called on every successful upsert.

    _normalizeToUtcMidnight: copy from repository impl (PATTERNS.md shared pattern).
