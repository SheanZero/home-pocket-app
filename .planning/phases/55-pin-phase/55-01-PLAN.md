Create the off-isolate salted-slow-hash KDF that is the SOLE brute-force defense for the 4-digit PIN (LOCK-07). Argon2id via the already-present `cryptography` 2.9.0 (NO new dependency), run inside `Isolate.run`, encoded as a PHC-style string stored in the EXISTING greenfield `StorageKeys.pinHash` keychain slot.
