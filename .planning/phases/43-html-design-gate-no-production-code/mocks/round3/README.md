## 应用的 5 条修改（三套一致）

1. **支出趋势移至顶部** —— 现为第一节，位于分类支出之上。
2. **支出趋势加 pill-tab：总支出 / 日常 / 悦己** —— 圆角胶囊，图标 + 文字，恰好一个 active（若叶绿 `#6FA36F` 实心 + 浅色文字；悦己 tab active 用樱粉呼应单月语义），inactive = 幽灵描边 + 灰字。**纯 CSS radio `:checked` 切换，无 JavaScript、无 `<script>`**，预绘 3 套 SVG 折线：
   - **总支出**：本月（实线绿）vs 上月（虚线灰）¥248,600 / ¥232,000。
   - **日常**：本月 vs 上月（日常部分）¥201,400 / ¥190,000。
   - **悦己**：**本月单线（樱粉），无上月/任何过往线** —— 见下「悦己 tab 单月决定」。
   - x 轴 7/14/21/28 日，中性标签（本月/上月），无评判措辞。
3. **小确幸日历热力移至合并分类节之后** —— 顺序：趋势 → 合并分类 → 日历 → 满足度直方图。
4. **合并「悦己花在哪」进「分类支出」** —— 见上 3 方案（A/B/C）。
5. **移除「悦己占比」ambient stat** —— 整块删除。

另：移除 V2 独立的「悦己分类构成 mini-bar」（已并入合并分类节，不再单列）。

## 悦己 tab「只显示本月一条线」的决定（ADR-012）

- 总支出、日常两个 tab 画 **本月 vs 上月** 两条线 —— 这是 Round-2 用户批准、记录在案的**支出侧实用例外**（与首页 `支出趋势` 对齐，标签中性、无「超过/落后/目标」措辞）。
- 悦己 tab **只画本月一条樱粉线，没有上月线、没有任何跨期对比** —— 悦己跨期对比 = 成就压力，是真正的毒性（ADR-012）。该 tab 附一句说明：「悦己只显示本月一条线 · 不与上月或任何过往比较（这是刻意的：为自己花的钱不该被拿来较劲）」。
- 由此，「上月」在悦己 tab 里只以**否定形式**出现（"无上月对比" / "不与上月"），绝无真实的上月数据线。

## 约束校验（已逐项渲染确认）

三套均通过 headless Chrome 渲染验证（视口 430×932，full-page 截图 + `evaluate_script` 探针）：

- **自包含**：`<script>` 标签数 = 0；外部 `link/img/iframe` = 0；无 CDN / 外部 CSS·JS·字体 / 网络。唯一的「`<script>`」「悦己占比」字样出现在说明性注释里（自包含声明 / 修改清单），非真实元素。
- **pill-tab 顶部 + 纯 CSS 切换**：默认 `总支出` 面板 `display:block`、其余 `none`；点 `日常`→只日常可见；点 `悦己`→只悦己可见。切换全靠 radio `:checked` 兄弟选择器，零脚本。
- **悦己 tab 单线**：悦己面板 `polyline` 数 = 1（单条樱粉线）、`.tl.prev`（上月图例）数 = 0。
- **甜甜圈/条几何**：A 主环 = 11 个 circle（1 外底 + 7 外弧 + 1 内底 + 2 内弧）；B 主环 8 + mini 抽屉 5；C 堆叠条 7 段求和 100%、悦己组 4 行小计 ¥47,200、辅助小环 3 circle。金额 JPY `¥` 无小数、tabular。
- **节序**：趋势 → 分类 → 小确幸日历 → 悦己满足度分布（三套一致）。
- **悦己占比已移除**：`.share`/`.share-note` 块数 = 0。
- **悦己侧无毒性**：零目标/进度环、零跨期、零排名（无「最棒/排名第一」）、零连续打卡/streak、零成就框定 —— 悦己全部为描述性 / 庆祝过去。HomeHero 悦己目标环隔离锁点（ADR-016 §3 / GATE-01）未被复制。
