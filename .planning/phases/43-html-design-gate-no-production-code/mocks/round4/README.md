## 本轮唯一改动 vs round3 B

把悦己抽屉里的 **mini-DONUT 子环** → 换成一条 **横向堆叠分段条（horizontal stacked segmented bar）**。

**理由（用户请求）：** 悦己可有很多类别；横条比环更能容纳「很多类」、更易读。横条仍读作主环樱粉悦己弧的「放大」（连接器 chip「把悦己这一块放大看看」保留）。

**除此之外，所有内容逐字等价保留**（行为不变）：

- 顶部「支出趋势」节 + 纯 CSS `:checked` pill-tab（总支出/日常 = 本月 vs 上月；**悦己 = 本月单线，无上月 / 无跨期**）。
- 主分类支出 donut hero（¥248,600，7 分类，悦己弧樱粉 `#D98CA0` 加粗强调）。
- 7 分类图例（实用）。
- 小确幸日历热力 **在合并分类节之后**（顺序：趋势 → 分类 → 日历 → 满足度直方图）。
- 满足度直方图（中位 7）。
- **无「悦己占比」** ambient stat。
- 自包含：内联 `<style>` + CSS radio `:checked`，**无 `<script>`、无外部 CSS/JS/字体 CDN/图片/网络**。

> 注：mini-ring 相关 CSS（`.mini-ring` / `.mc` / `.drawer-body` flex 布局 / `.dl` 旧 donut 图例）已移除，替换为 `.joybar` / `.joybar .seg` / `.joybar-legend` / `.jl`。抽屉外壳（`.drawer` / `.joy-connector` / `.drawer-cap2`）保留。

## 悦己新分类调色板（暖色、色相分离）

旧版 mini-donut 用的是 **同一樱粉色相的明度梯度**（`#D98CA0 → #E6A6B8 → #EFC0CC → #F4D6DE`），相邻段难以分辨。
新条采用 **暖色系分类调色板**，逐段「色相」明显分离，锚定樱粉，向琥珀/金、珊瑚/赤陶、梅紫/藕、桃/沙、暖灰褐、藕灰玫展开。
**刻意避开 ADR-019 的若叶绿 `#6FA36F`/`#5FAE72` 与钢蓝 `#5B8AC4`**——那是日常/共享账本的语义色，悦己子类用了会串语义线。

- **段间分隔：** 2px 奶白 `#FFFAF6`（`--cream`），段右 `border-right`，最后一段去边——边界清晰。
- **相邻色相对比：** 樱粉→琥珀金→珊瑚→梅紫→桃沙→暖褐→藕灰玫，每对相邻段色相明显不同（headless 实测 7 段背景色全唯一）。

- 主环悦己弧、悦己 tab、日历热力、直方图仍用 ADR-019 主樱粉 `#D98CA0`/joyText `#A53D5E`——分类条调色板只作用于「悦己子类」这一处。

## 横条实现（纯 CSS，自然合计 100%）

- 一条满宽 flex 容器（`.joybar`，高 32px，圆角 9px，`overflow:hidden`）。
- 每段 `flex-grow: <金额>`（12000 / 10400 / 8800 / 6300 / 4500 / 3400 / 1800）——flex 按比例分配宽度，**合计天然 = 100%**，无需手算像素。
- largest→smallest 排序。前 4 大段内嵌轻量 `%` 标（白字 + 文字阴影）；后 3 小段不挤标，靠图例读数。
- 条下图例 2 列网格（`.joybar-legend`）：每行 = 色块 + 类名 + 金额(¥) + 占比(%)。

## ADR-012 反游戏化合规（悦己侧）

横条框定语全程**描述性**，零毒性：

- 标题：「悦己 ¥47,200 花在哪几类开心事」+ 副标「仅呈现去向，不分高下」。
- 脚注：「百分比是各项占『悦己自身 ¥47,200』的比例 · 仅呈现去向，不分高下、不设目标、不与过往比较」。
- **无排名**（无「最棒/排名第一」等竞争措辞；largest→smallest 仅为视觉排布，不标名次）。
- **无目标 / 无进度环**（横条是构成占比，不是 target/progress；¥47,200 是绝对累计，无基线对比）。
- **无跨期**（悦己侧零本月 vs 上月；跨期仅在 总支出 / 日常 两个支出 tab 出现）。
- **无连续打卡 / streak**（日历热力仍只看「哪些天发生过」，不数连续）。
- **无成就框定**。

范围合规：不画入账侧 / 不画结余比率；家庭数据仅聚合（本稿为个人视图）。
