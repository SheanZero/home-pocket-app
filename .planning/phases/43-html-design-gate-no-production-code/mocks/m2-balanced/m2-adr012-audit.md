# ADR-012 自审表 — Mock M2 均衡（balanced）

**Phase:** 43-html-design-gate-no-production-code（GATE-02）
**写于:** 2026-06-15
**判定标准:** RESEARCH §4 反游戏化映射表 + ADR-012《Forbidden Features》#1–#6 + ADR-016 §3/§5。
**判定口诀（RESEARCH §4）:** 动画/强调若由「view/data 出现」触发 = OK ✅；若由「跨过阈值/目标/连续」触发 = forbidden ❌。

> M2 是 5 套里**悦己浓度中等**的一案：实用区（支出总览 + donut + breakdown + 6 月趋势柱）与悦己区（值得卡 + **满足度直方图** + **记忆故事条**）**等重**。
> 因为 M2 真正画出了满足度直方图与记忆故事条——这两个元素正是反游戏化最容易越界的地方——本表对它们做**专项复核**：
> 直方图必须是**分布展示**（不得标「超过上月 / 目标 8+」），故事条必须是**单条叙事回顾**（不得做「最棒悦己分类」排名）。逐元素填写，任何 ❌ 须在 GATE-03 选定前移除或降级。

## 二、关键红线复核（M2 专项 —— 直方图 + 故事条重点）

- **满足度直方图无 vs / 无目标:** 直方图只画 1–10 分布 + 中位 7 pill + 描述性 caption。**无**「超过上月」「目标 8+」「应达到」「比上月更满意」任何措辞或视觉箭头。中位描边仅定位，非「达标线」。✅（专项复核通过）
- **记忆故事条是叙事非排名:** 故事条呈现**单条**最有温度的悦己瞬间（外出·体验 ¥4,200，2026/05/18，满足 10/10，一句记忆）。标题「这个月最有温度的一笔」是回顾叙事，**非**「最棒悦己分类」排名、**非**「第一名」、**无**奖杯/勋章。满足 10/10 是该笔的事实属性，非「最高分纪录/连续满分」。✅（专项复核通过）
- **无进度/目标环:** analytics 侧零 progress/target ring；唯一 target ring 归 HomeHero 独占（ADR-016 §3）。值得卡是大号**数字 + 文字**，绝非环。✅
- **仅支出侧:** 全页只画支出（总览 / donut / breakdown / 趋势 / 悦己累计）——无收入录入、无结余比率（范围勘误 INCOME-V2-01；GATE-01 deep-map「范围勘误」）。✅
- **无跨期对比:** 趋势柱是中性滚动上下文，当前月仅以浅描边定位，无任何「vs 上月」措辞或视觉箭头。✅
- **无排名 / 无 best-分类:** breakdown 按金额自然降序是**信息排序**，非「冠军/最棒」竞争框定；故事条单条叙事非分类排名。无「第一名」高亮、无奖杯/徽章。✅
- **calm-warm register（D-04）:** 悦己文案「每一笔，都是给自己的一点温柔。值得。」「也都是真实的体验。」是平静肯定、像日记；无打分、无外放、无「最高/达成/连续/超过/目标/最棒」红线词（RESEARCH §3b）。✅
- **自包含 / 零网络:** 内联 `<style>`，无外部资源 / 无脚本 / 无网络字体——符合零知识 app 气质（威胁 T-43-03）。✅

## 三、整套裁定

**整套裁定: PASS**

M2 作为悦己浓度中等的均衡方向，**真正画出了**满足度直方图与记忆故事条这两个高风险元素，并经专项复核确认：直方图是纯分布展示（无「超过上月 / 目标 8+」框定），故事条是单条叙事回顾（无「最棒分类」排名）。全部情感元素均落在 ambient / 庆祝过去（OK ✅）一侧，**零遗留 ❌**。无进度/目标环、无跨期对比、无排名、无成就/徽章/连续，仅支出侧，calm-warm 文案。可直接进入 GATE-03 评判，无需调整。

> 任何 ❌ 必须在选定前移除或降级为 ambient——本表无 ❌，无需处置。
