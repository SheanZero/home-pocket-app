## D-11 判定矩阵（选定唯一一案）

**首要标准（进入评判核心）:**

| 维度 | 含义 | 怎么看 |
|---|---|---|
| **悦己情感共鸣** | 「为自己花钱而开心」是否真被表达出来——calm-warm、像日记、有温度 | 看悦己端读感：是否打动、是否克制不外放、是否"想再看一眼" |
| **实用性** | 支出总览/趋势/分类下钻是否清晰可用，信息密度是否合适 | 看实用侧 IA：能否快速回答"这月花在哪、花了多少" |
| **ADR-012 安全度** | 每个情感元素是否守住 ambient/庆祝过去，无目标/跨期对比/成就/排名 | 看该套 `adr012-audit.md` 自审表是否**零遗留 ❌** |

**次要上下文（参考，不进首要标准）:**

| 维度 | 含义 |
|---|---|
| 复用度 | 多大程度复用 17 个现成 widget（重组 vs 重画） |
| 低成本 | 落地工作量、对 Phase 44-46 的改造负担 |

> 复用度/低成本是**次要上下文**——帮助理解落地代价，但**不决定**选哪一案。选案由「悦己情感共鸣 / 实用性 / ADR-012 安全度」三项首要标准定。
