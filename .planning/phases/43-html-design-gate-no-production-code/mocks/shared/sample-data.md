# 共享示例数据 — 模拟一家庭一月账目（5 套 mock 共用）

**Phase:** 43-html-design-gate-no-production-code
**写于:** 2026-06-15
**用途:** D-09 真实感示例数据。**这是全部 5 套 mock（M1..M5）唯一共用的数据集**——所有 mock 用同一组数字，保证评判时差异只来自**设计**，不来自数据（可比性）。

> ⚠ **全部数字均为虚构（FICTIONAL / SIMULATED）。** 这是为设计探索人为编造的一家庭一月账目，**绝非任何真实用户的财务数据**。这是零知识 app 的安全红线（威胁 T-43-01）：示例数据必须是模拟数据，永不嵌入真实账目。
>
> **货币:** 全部金额为 **JPY 整数，¥ 无小数**（ADR-019 amount style，如 `¥1,235`）。

| 字段 | 值 |

## 6. 家庭模式 aggregate 数（仅聚合，无 per-member 行）

`FamilyHappiness` 类型契约：**只有聚合字段，没有 per-member 字段**（ADR-012 §6）。mock 家庭模式**只能画下表的聚合数，绝不画成员排行 / 成员贡献 / 成员排序列表**。

| 聚合字段（aggregate-only） | 值 |
|---|---|
| `familyHighlightsSum`（家庭悦己亮点累计） | ¥73,800 |
| `medianSatisfaction`（家庭中位满足度） | 7 |
| `totalGroupJoyTx`（家庭悦己笔数合计） | 35 |
