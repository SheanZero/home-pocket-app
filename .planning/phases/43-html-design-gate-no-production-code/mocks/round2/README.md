## 每套悦己图表调色板（来自指令给定的「悦己图表 palette」）

所有悦己图表均为**描述性 / 庆祝已发生**，无 target/goal/progress/ranking/streak/cross-period。

| 悦己图表 | V1 | V2 | V3 | 性质 |
|---|:--:|:--:|:--:|---|
| 满意度 1–10 分布直方图 | ✅ | ✅ | ✅ | 描述形状（中位 7 标注，无目标 8+） |
| 悦己分类构成（钱花在哪） | mini-bar | mini-bar | 小环 + 明细列表 | 描述去向，不排名 |
| 小确幸日历热力 | — | ✅ | ✅ | 庆祝月的纹理（哪些天有悦己一刻），**不计连续天数** |
| 悦己占总支出比（ambient） | — | ✅ | ✅ | 描述既成事实「占 19%」，非目标 X% |
| 悦己累计微曲线（单月） | — | — | ✅ | 月内累计走势，**无目标线、无上月线**（区别于支出侧实用例外） |

## ADR-012 反游戏化安全声明（三套统一）

- **悦己侧:** 零跨期对比、零目标/进度环、零排名（无「最棒分类」）、零连续打卡 / streak、零成就框定。悦己 = 庆祝已发生、环境化、描述性。
- **支出侧「本月 vs 上月」趋势 = 用户批准的实用例外**（与首页支出趋势的预算对照一致），保持中性描述，不带「超过/落后/目标」判定。此例外**仅限支出侧**；悦己累计微曲线（V3）严格单月、无上月线、无目标线。
- **家庭数据仅聚合**（本轮三套为个人视图，未画家庭；若入家庭视图，仅聚合字段，绝无 per-member 行）。
- **不画入账侧 / 不画结余比率**（GATE-01 范围勘误）。
- 分类环把 4 个悦己分类滚为「悦己合计」一段（桜粉），与日常各分类同台；悦己内部 breakdown 仅在悦己区按金额呈现去向，不与日常比、不排名。

> ADR-016 §3/§4 的「悦己目标进度环」是**首页 HomeHero** 的元素，**不**在统计页复制——统计页悦己一律描述性。
