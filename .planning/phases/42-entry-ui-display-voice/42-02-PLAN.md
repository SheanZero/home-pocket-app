Establish the shared decimals foundation that BOTH the keypad/display wave and the voice wave depend on: route per-currency minor-unit decimals through `intl 0.20.2`'s authoritative `currencyFractionDigits` map (per RESEARCH Q1) instead of the current hardcoded `default: 2`. Purpose: One authoritative decimals source so the AmountInputController cap (D-07), the truncation target (D-08), list/preview formatting (DISP-02), and stored subunits all agree.
Output: Extended `number_formatter.dart` + `currency_conversion.dart`.
