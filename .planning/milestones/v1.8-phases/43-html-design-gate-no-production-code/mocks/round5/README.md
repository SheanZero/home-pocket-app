# Round 5 — `r5-drawer-joybar.html`（浅色 / LIGHT MODE ONLY）

派生自 `round4/r4-drawer-joybar.html`。本轮只做 **3 处精确改动**，其余行为逐字等价保留
（顶部「支出趋势」纯 CSS pill-tab、主分类 donut hero、悦己横向堆叠条抽屉、小确幸日历在分类节之后、
满足度直方图、无「悦己占比」、自包含内联 `<style>`、CSS radio `:checked` 切换、无 `<script>`、无外部引用）。

## 3 处改动 vs round4

1. **悦己抽屉图例 → 单列。**
   `.joybar-legend` 由 `display:grid; grid-template-columns:1fr 1fr`（2 列）改为
   `display:flex; flex-direction:column`（单列）。7 个悦己子类逐行纵向排列，每行 = 色块 + 类名 + ¥金额 + 占比%。
   横向堆叠条本身与 7 色暖调色板（`--j1..--j7`）**不变**。

2. **主分类支出节 → 严格按金额降序、仅 level-1（1 级）分类。**
   donut 弧段顺序 **和** 图例列表都按 ¥ 从大到小排列；扁平列表，**无子分组 / 无嵌套 / 无 日常·悦己 父级表头**。
   10 个 1 级分类（占比基于总支出 ¥248,600，数据见 `../shared/sample-data.md §2`）：

   | # | 分类 | 金额 | 占比 |
   |---|---|---|---|
   | 1 | 饮食 | ¥78,300 | 31.5% |
   | 2 | 住居 | ¥62,000 | 24.9% |
   | 3 | 交通 | ¥21,800 | 8.8% |
   | 4 | 日用品 | ¥18,500 | 7.4% |
   | 5 | 书籍·学习 | ¥16,400 | 6.6% |
   | 6 | 兴趣·爱好 | ¥14,900 | 6.0% |
   | 7 | 通信 | ¥11,200 | 4.5% |
   | 8 | 外出·体验 | ¥9,700 | 3.9% |
   | 9 | 其它 | ¥9,600 | 3.9% |
   | 10 | 美容·放松 | ¥6,200 | 2.5% |

   合计 = ¥248,600 ✓ ；占比合计 = 100.0% ✓
   donut 弧 dasharray = 金额 / 248600 × 周长(502.6548)，offset 累加，与列表同序。

3. **移除主分类节里的「悦己合计」聚合。**
   删去 round4 图例里的 `悦己合计 ¥47,200 19.0%` 行，以及环中心的 `日常 81% · 悦己 19%` 拆分线（`.ctr .splitline` 类 + CSS 整段移除）。
   donut **中心只显示总计 `¥248,600` + 中性标签「本月支出」+ `86 笔`**，主分类统计里**不出现任何悦己聚合数**。
   各分类着色保留——4 个悦己系分类（书籍·学习 / 兴趣·爱好 / 外出·体验 / 美容·放松）仍用樱粉 `--cat-joy` 作账本暗示，但只是单分类着色，不是聚合。
   下方独立的「悦己花在哪」抽屉**保留**自己的 `¥47,200` 上下文（那是悦己明细节本身，不是主分类统计）。
