# ADR-012 自审表 — Mock M4 温暖反思派（warm-reflective joy-led）

**Phase:** 43-html-design-gate-no-production-code（GATE-02）
**写于:** 2026-06-15
**判定标准:** RESEARCH §4 反游戏化映射表 + ADR-012《Forbidden Features》#1–#6 + ADR-016 §3/§5 + 43-CONTEXT.md D-05/D-06。
**判定口诀（RESEARCH §4）:** 动画/强调若由「view/data 出现」触发 = OK ✅；若由「跨过阈值/目标/连续」触发 = forbidden ❌。

> M4 是 5 套里**悦己中浓度**的一案，也是**最情感外放**的方向：悦己主导，情感核心（值得卡 + kakeibo Q4 反思 prompt + 满足度分布）**先行**，实用支出总览**退居次要**紧凑条。M4 的探索焦点是 D-03「**中强度**」JOY-01 读感（比 M3 更温暖、更 present——自信桜粉、字号偏大、柔光底，但**仍是一行肯定的数字，绝非环**），并且是 5 套中**kakeibo Q4 反思 prompt 的首要展示**（D-02：M4 为核心展示）。
> 因为 M4 引入了**唯一新增视觉元素**——kakeibo Q4 反思 prompt（不在现有 17 widget 中），本表的专项关切有二：(1) **中强度的值得数字是否仍读作 ambient（庆祝已发生的好），而非"朝目标累积的分数"**；(2) **Q4 反思 prompt 是否严格落在「开放/肯定/价值观」一侧，而非「评分式/目标式提问」，并且严格静态只读（D-06）不接受任何输入**。逐元素填写，任何 ❌ 须在 GATE-03 选定前移除或降级。

## 二、关键红线复核（M4 专项 —— 中强度值得数字 + kakeibo Q4 prompt）

- **中强度数字仍读作 ambient，非"朝目标累积的分数":** 值得卡呈现的 ¥47,200 是**绝对累计 Σ joy_contribution**（= 当月悦己支出累计），**无基线、无 target、无进度环、无跨期 delta**。中强度只改变它的**视觉权重**（38px、自信桜粉、柔光底、🌸 水印），不改变它的**语义**——它仍是「这个月你已经为自己花了多少」的事实陈述，注脚「无目标、无对比 · 只记录已经发生的好」显式锁死 ambient 框定。读者无法把它误读为"距某目标还差多少"或"本月攒到第几格"。✅（专项复核通过）
- **kakeibo Q4 反思 prompt 落在「开放/肯定/价值观」一侧，且严格静态只读:** Q4 提问「下次，什么会让你花得更开心一点？」是 kakeibo 原生第四问的非游戏化先例，框定为**价值观肯定**（"把钱多留给真正让你享受、回想起来仍觉得值得的事"，明示"不是要你少花"），**非评分式**（无"打几分"）、**非目标式**（无"下月目标定多少"）。视觉上是一句温柔提问 + 一行示例引导，**不接受任何输入**——无 `<textarea>`、无 `<input>`、无 `<button>`、无 submit；注脚「这是一句给你自己的提问 · 只供回味，不需要填写、不会被记录」显式锁死**不持久化**（D-06）。因此**不触发 JOY-04 用户文本持久化**（本里程碑 no-go，D-06/D-07），无加密/隐私含义。✅（专项复核通过）
- **满足度=分布+描述，非评分达标:** 直方图是 1–10 的分布形状（中位 7），配描述性反思（"大多落在满足的那一端"；"偶有一两笔不那么尽兴——那也没关系"）。**零**"超过上月"、"目标 8+"、"平均分达标"、"连续满分"措辞。高亮柱仅按分布密度自然着色（f(data)→color），非达标高亮/冠军格。✅
- **无进度/目标环:** analytics 侧零 progress/target ring；唯一 target ring 归 HomeHero 独占（ADR-016 §3）。值得卡是数字+文字，柔光底与 `.pip` 是固定装饰非进度填充。✅
- **仅支出侧:** 全页只画支出（值得累计 / 满足度 / 总览 / 分类）——无入账录入、无结余比率（范围勘误 INCOME-V2-01；GATE-01 deep-map「范围勘误」）。✅
- **无跨期对比:** M4 零趋势柱、零「vs 上月」措辞、零涨跌箭头（`previousMonthComparison` 已算但绝不 surface）。✅
- **无排名 / 无 best-分类:** 分类小览按金额自然降序是**信息排序**，非「冠军/最棒」竞争框定；M4 未独立画故事条，无任何"最高分/连续满分/最棒一笔"红线表述。✅
- **calm-warm register（D-04）:** 全文「都是给自己的一点温柔。值得。」「轻轻想一想」「那也没关系，都是认识自己的一部分」是平静温暖、像日记的措辞；无打分、无外放、无「最高/达成/连续/超过/目标/最棒/排名/分数」红线词（RESEARCH §3b）。即便 M4 是 5 套中最情感外放的方向，外放仅体现为**视觉温度与版面权重**，措辞仍严守 calm-warm。✅
- **自包含 / 零网络:** 内联 `<style>`，无外部资源 / 无脚本 / 无网络字体——符合零知识 app 气质（威胁 T-43-03）。✅

## 三、整套裁定

**整套裁定: PASS**

M4 作为悦己中浓度、最情感外放的反思方向，把情感核心（值得卡中强度 + kakeibo Q4 反思 prompt + 满足度分布）置于实用总览之前。两项专项复核均通过：(1) 中强度只改变值得卡数字的**视觉权重**（38px、自信桜粉、柔光底），**不改变其 ambient 语义**——它仍是绝对累计 Σ joy_contribution 的事实陈述，无基线、无 target、无进度环、无跨期 delta，注脚显式锁死「无目标、无对比 · 只记录已经发生的好」；(2) kakeibo Q4 反思 prompt 严格落在「开放/肯定/价值观」一侧（非评分式、非目标式），并**严格静态只读**（无 textarea/input/button/submit，注脚明示"不需要填写、不会被记录"），因此不触发 JOY-04 持久化（本里程碑 no-go，D-06）。满足度为分布+描述（非达标评分），全页仅支出侧，无跨期对比、无排名、无成就/徽章/连续，calm-warm 文案。全部情感元素均落在 ambient / 庆祝过去（OK ✅）一侧，**零遗留 ❌**。可直接进入 GATE-03 评判，无需调整。

> 任何 ❌ 必须在选定前移除或降级为 ambient——本表无 ❌，无需处置。
