## 这是什么

这是 GATE-03「选定恰好一案」闭合的**最终方向**：用户在 round5 批准的
`round5/r5-drawer-joybar.html`（变体 B「主环 + 悦己横向堆叠分段条」，沿 M1–M5 强度轴的 **M2 均衡**衍生）。

本目录把它定稿为可交付的三件套：浅色（含一处数据修正）+ 深色 + ADR-012 自审。

## 数据修正（vs round5 · 唯一改动）

**悦己子类现为「主 level-1 分类」的严格子集（subset），每段 ≤ 该分类在主环里的金额。**

round5 的悦己条里列了「旅行」「美食犒赏」——这两类不在主 10 类清单内，自相矛盾。本稿把悦己
¥47,200 重分配到确实出现在主环里的分类上：

| 悦己子类 | 金额 | 占悦己 % | 主分类总额 | ≤ 主总额 |
|---|---|---|---|---|
| 兴趣·爱好 | ¥14,900 | 31.6% | ¥14,900 | ✓（全额悦己）|
| 书籍·学习 | ¥13,000 | 27.5% | ¥16,400 | ✓ |
| 外出·体验 | ¥9,700 | 20.6% | ¥9,700 | ✓（全额悦己）|
| 美容·放松 | ¥6,200 | 13.1% | ¥6,200 | ✓（全额悦己）|
| 饮食（犒赏）| ¥2,400 | 5.1% | ¥78,300 | ✓ |
| 日用品（犒赏）| ¥1,000 | 2.1% | ¥18,500 | ✓ |
| **合计** | **¥47,200** | **100.0%** | | ✓ |

色彩一致性：悦己身份系分类锚定主环悦己弧樱粉色系，每段一个独立暖色相（避开绿/蓝日常·共享语义色）；
「犒赏」小额用暖色相区分。

其余一切（趋势三 tab、主环 hero、日历、直方图、文案、节顺序）逐字等价保留。

## 跨期例外（带入 GATE-04）

**跨期（本月 vs 上月）仅出现在 `支出趋势` 的 总支出 / 日常 两个支出侧 tab** —— 这是
`round2/ROUND2-DECISION.md` 记录的 **用户批准的 ADR-012 §4 例外**（实用预算 parity，与首页支出趋势对齐，
标签中性）。**悦己 tab 与所有悦己元素 zero 跨期 / zero 目标 / zero 进度环 / zero 排名 / zero 连续打卡。**
此例外须在 **GATE-04 ADR go/no-go** 中作为记录在案的决定纳入，非静默违规。详见 `selected-adr012-audit.md`。
