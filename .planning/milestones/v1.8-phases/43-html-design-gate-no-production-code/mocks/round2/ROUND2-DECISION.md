## Brainstorm 出的悦己图表候选池（全部 ADR-012 安全：描述性/庆祝过去，无目标·无进度·无跨期·无排名·无连续打卡）

- 满足度 1–10 分布直方图（描述形状）
- 悦己分类构成（钱花在哪类开心事 —— 仅呈现去向，不分高下、不排名）
- 小确幸日历热力（本月哪些天有悦己一刻 —— 庆祝月的纹理，**不计连续天数/streak**）
- 悦己占总支出比（单一环境化描述数字，**非目标 X%**）
- 悦己累计微曲线（**单月**累计，**无目标线、无上月线/跨期**）
- 平均/中位满足度（calm 描述，**无「目标 8+」**）
