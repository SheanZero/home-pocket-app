    - `feedChunk(String text, {required bool isFinal})`:
      - If `!isFinal` → return (partials ignored).
      - If `text.isEmpty` → return.
      - If buffer is empty (first final this session): buffer = text, set _lastFinalAt = clock.now(), start 2.5s timer, call speechService.restartListen() (await; propagate errors), return.
      - Else: evaluate double gate (time + lexical) on (buffer, text, clock.now()):
        - Pass: append text to buffer (no separator — RESEARCH §Pattern 3 shows direct concat); reset _lastFinalAt = clock.now(); cancel and restart 2.5s timer; await restartListen().
        - Fail: commit current buffer (call parser.parse, emit via onAmountResolved if non-null), clear buffer + timer + lastFinalAt. Do NOT seed a new buffer with the rejected chunk (per RESEARCH §Pattern 3 walk-through "现金" case line 481 recommendation: "do not start a fresh buffer with the non-numeric chunk"). Do NOT call restartListen() — recording session ends from merger's perspective; screen owns the actual stop.
    - `stop()` (user-initiated): commit buffer immediately (if non-empty) → parse → emit if non-null → clear all state. Does NOT call recognizer stop (that's the screen's job).
    - `dispose()`: cancel timer, clear buffer + lastFinalAt. Does NOT emit any pending buffer (synchronous teardown). Idempotent.

    Lexical gate predicate per RESEARCH lines 432-461:
    - `_bufferLooksOpen(buffer)`:
      - normalize(buffer) → tokens; if empty → false.
      - Case A: last token is `Unit` with `power >= 100` (千/百/万) → true. Speaker likely to add tens digit.
      - Case C: last token is `Digit` AND the previous Unit (anywhere in the section) has `power >= 100` → true. Bare-tail digit after a larger unit — possibly incomplete.
      - Otherwise → false. Includes: last is Unit(10) alone, last is Skip (shouldn't happen since normalize drops Skip), last is ZeroPlaceholder (shouldn't be last in a sensible amount), last is Digit with only smaller units before it.
    - `_chunkStartsNumeric(chunk)`:
      - normalize(chunk) → tokens; if empty → false.
      - First token is `Digit` or `Unit` (or `PackedToken` — flatten or treat as numeric leader) → true.
      - First token is `ZeroPlaceholder` → false (zero alone isn't a continuation that the speaker meant — leading zero in a continuation is implausible).

    Time gate:
    - `now.difference(_lastFinalAt!) <= Duration(milliseconds: 2500)` → pass.

    Constants (file-scope private):

       ```dart
       class VoiceChunkMerger {
         VoiceChunkMerger({
           required NumeralStateMachine parser,
           required SpeechRecognitionService speechService,
           required AmountResolvedCallback onAmountResolved,
           DateTime Function()? clock,
         })  : _parser = parser,
               _speechService = speechService,
               _onAmountResolved = onAmountResolved,
               _clock = clock ?? DateTime.now;

         static const _windowDuration = Duration(milliseconds: 2500);

         final NumeralStateMachine _parser;
         final SpeechRecognitionService _speechService;
         final AmountResolvedCallback _onAmountResolved;
         final DateTime Function() _clock;

         String _buffer = '';
         DateTime? _lastFinalAt;
         Timer? _windowTimer;

         Future<void> feedChunk(String text, {required bool isFinal}) async {
           if (!isFinal) return;
           if (text.isEmpty) return;

           final now = _clock();

           if (_buffer.isEmpty) {
             _buffer = text;
             _lastFinalAt = now;
             _restartWindowTimer();
             await _speechService.restartListen();
             return;
           }

           if (_shouldMerge(_buffer, text, now)) {
             _buffer += text;
             _lastFinalAt = now;
             _restartWindowTimer();
             await _speechService.restartListen();
           } else {
             // Fail: commit existing buffer; drop new chunk (per RESEARCH §Pattern 3
             // walk-through "现金" case — non-numeric continuation breaks the window).
             _commitAndClear();
             // Note: per CONTEXT.md D-09, the screen owns the actual stop()/cancel()
             // on the recognizer. Merger does NOT call stopListening here — the
             // failed-merge chunk just gets dropped from the merger's perspective.
           }
         }

         /// User-initiated commit (typically called when the screen's tap-to-stop fires).
         void stop() {
           _commitAndClear();
         }

         void dispose() {
           _windowTimer?.cancel();
           _windowTimer = null;
           _buffer = '';
           _lastFinalAt = null;
         }

         // ─── Internals ───

         bool _shouldMerge(String buffer, String chunk, DateTime now) {
           if (_lastFinalAt == null) return false;
           if (now.difference(_lastFinalAt!) > _windowDuration) return false;
           if (!_bufferLooksOpen(buffer)) return false;
           if (!_chunkStartsNumeric(chunk)) return false;
           return true;
         }

         bool _bufferLooksOpen(String buffer) {
           final tokens = _parser.normalize(buffer);
           if (tokens.isEmpty) return false;
           final last = tokens.last;
           if (last is Unit && last.power >= 100) return true;
           if (last is Digit) {
             final units = tokens.whereType<Unit>().toList();
             if (units.isNotEmpty && units.last.power >= 100) return true;
           }
           return false;
         }

         bool _chunkStartsNumeric(String chunk) {
           final tokens = _parser.normalize(chunk);
           if (tokens.isEmpty) return false;
           final first = tokens.first;
           return first is Digit || first is Unit || first is PackedToken;
         }

         void _restartWindowTimer() {
           _windowTimer?.cancel();
           _windowTimer = Timer(_windowDuration, _commitAndClear);
         }

         void _commitAndClear() {
           final pending = _buffer;
           _windowTimer?.cancel();
           _windowTimer = null;
           _buffer = '';
           _lastFinalAt = null;
           if (pending.isEmpty) return;
           final amount = _parser.parse(pending);
           if (amount != null) {
             _onAmountResolved(amount);
           }
         }
       }
       ```
