<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| User input -> SettingsRepository | User-entered integer becomes persisted app preference |
| Recommendation provider -> Settings UI | Historical aggregate recommendation becomes visible reference |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-14-04-01 | Tampering | Target input | mitigate | Reject zero, negative, decimal, and non-integer input before persistence |
| T-14-04-02 | Information disclosure | Recommendation copy | accept | Displays only aggregate target integer, not month history or transactions |
| T-14-04-03 | Social/behavioral harm | Recommendation framing | mitigate | Copy guard forbids comparison/delta/achievement framing |
| T-14-04-04 | Elevation of privilege | Persistence layer | accept | Uses existing local SettingsRepository only; no new storage privilege or network path |
</threat_model>
