<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| ShoppingItem.note (plaintext) → ShoppingItemRow.note (ciphertext) | Encryption enforced at this boundary; never stored plaintext |
| ShoppingItemRow.tags (JSON string) → ShoppingItem.tags (List<String>) | JSON decode with try/catch prevents malformed data from crashing the stream |
| FieldEncryptionService decrypt failure → note = null | Silent failure prevents ciphertext leakage via logs |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-36-11 | Information Disclosure | `note` plaintext at rest | mitigate | `FieldEncryptionService.encryptField` called before every insert/update/upsert; decryptField called on every read; catch block silences errors without logging ciphertext |
| T-36-12 | Tampering | tags JSON injection | mitigate | `jsonEncode(List<String>)` from `dart:convert`; decode wraps in `try/catch` returning empty list on malformed input |
| T-36-13 | Information Disclosure | decrypt failure leaks note content | mitigate | `catch (_) { decryptedNote = null; }` — empty catch, no `e.toString()` or `debugPrint` |
| T-36-SC | Tampering | npm/pip/cargo installs | accept | No new packages |
</threat_model>
