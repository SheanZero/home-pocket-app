悦己充盈环 · 配色方案探索

固定 ② 扫掠尾迹（comet）+ ⑥ 充盈光晕（glow）两种过渡。本轮转向 浅色系： 不用任何红 / 玫 / 品红，全部柔和粉彩，延续 app 的 Teal Clarity 清透气质。三套都在 安全色族（青 / 绿 / 蓝 / 薰衣草 / 浅金）里选色，色相 + 明度拉开；内环统一用暖浅金呼应"悦己"。各 ②/⑥ 两版对照。

三色映射：外环 小确幸 · 中环 满足度 · 内环 悦己目标（= 圆心悦己指数色）。 三环相切无间距、圆头端点、OKLCH 环内插值；三色按色相 + 明度双重拉开 → 既解决"中外环太近"，也过色盲与小尺寸。 Pencil 当前环境无法落盘（D-03b），故以可入库的 HTML/SVG 交付，已按真机尺寸 + 手机外框呈现。
