下段「本月最爱」ストリップ／悦己 pill／金額 が実際の波及面。各候補は 日常(teal) と横並びで対比。

推奨の見立て（teal アンカーとの相性 × 悦己の"自分へのご褒美"らしさ）

  • テラコッタ / 赤陶 — teal の定番相方。暖かく落ち着いた高級感、黄より大人。最も安全な"映える暖色"。
  • 蘭紫 / Orchid — teal の補色寄りで最もコントラストが立つ。digital-lavender 的な wellness/self-care 感が悦己テーマに直結。"脱・黄色"を最も強く出せる。
  • 玫红 / Magenta — 最も祝祭的で華やか。teal と冷暖で強対比。ただし error 赤との距離に注意。
  • 杏橙 / Tangerine・珊瑚 / Coral は暖色だが赤/旧コーラル identity に寄るため中庸。莓紫 / Plum は上品だが沈みがち。
