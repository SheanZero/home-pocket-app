悦己 (Joy) 配色探索 teal #1C7A86 と対で映える identity 色を探す｜赤は error 専用で除外・黄/金は不満につき除外｜金額テキストは白カード上 WCAG AA ≥4.5:1 必須

下段「本月最爱」ストリップ／悦己 pill／金額 が実際の波及面。各候補は 日常(teal) と横並びで対比。

推奨の見立て（低彩度・くすみ系 — teal と静かに対比しつつ大人びた悦己感）

  • 霧紫 / Dusty Lavender — 残した Orchid を素直に脱彩度した版。「蘭紫は良いが派手」への直球の答え。上品で teal と静かに対比、wellness 感も維持。
  • 陶土 / Muted Clay — Terracotta のくすみ版。最も大人で温かく、teal と古典的相性。低彩度の暖色枠の本命。
  • 丁香 / Mauve・豆沙粉 / Dusty Rose — 紫〜ピンクのくすみ。落ち着いた華やぎ。甘すぎず error 赤とも分離。
  • 柔珊瑚 / Dusty Coral は暖かいが赤寄り、紫藤 / Heather は最も静かだが teal・shared 青に寄るので個性は控えめ。
  • ※ くすみ色は彩度が低いぶん、金額テキスト(joyText) は AA を満たすよう各候補とも濃いめに振っています（各列の AA 比を参照）。
