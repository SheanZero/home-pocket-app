# Home Pocket V16 Mockup

V16 由 V15 整 App Mockup 升版而来，并于 2026-07-16 与当前 Flutter App 的主页面及统一记账实现重新同步。

## 打开方式

直接在浏览器打开 [index.html](index.html)。Mockup 为单文件交互稿；Material Symbols 图标字体通过 Google Fonts 加载。

## V16 同步内容

- 主页、明细、统计、购物统一使用同一主标题栏规范：顶部 11px、左右 20px、标题栏高度 46px、标题 20/28/700、标题栏与正文间距 13px。
- 主标题在左；右侧操作按钮为 40×40，图标为 24px。主页、明细、统计保留日历与设置，购物页补齐设置入口。
- 主页模式徽标同步为 27px 最小高度、11/15/700 字体与 15px 图标。
- 购物筛选恢复“私有”和“类目”：个人模式显示账簿段控件与二级筛选；家庭模式额外显示“全部 / 个人”范围段控件。
- 私有、类目和账簿筛选可以组合；类目使用购物专用的 L1 多选底部面板，支持清空、取消和应用。
- 家庭范围切换会重置账簿、私有与类目筛选，与 App 状态逻辑一致。
- 手工与语音记账改为同一共享表单：金额、类目、日期、商家、用途和备注不换页，语音面板只替换底部数字键盘。
- 统一记账提供 A「原位替换」、B「双轨提示」、C「金额旁快捷」三种可切换设计；默认 A 最贴近当前 `ManualOneStepScreen`。
- 语音演示覆盖聆听、解析、识别结果和麦克风不可用；停止、恢复原内容、返回表单均使用明确按钮，不再依赖“点击空白处退出”。
- 语音填充的字段显示来源标记；低置信度类目显示“要确认”，确认前禁用记录按钮。金额在手动与语音状态中始终使用同一份数据。

## 核验状态

- A1：个人 · 浅色
- A2：家庭 · 浅色
- A3：个人 · 深色
- 视口快捷宽度：375 / 390 / 430
- 统一记账方案：A / B / C；演示状态：手动 / 聆听 / 识别结果 / 麦克风不可用

V16 是当前整 App Mockup 的唯一实现基准；V15 及其重复副本已由该版本取代。
