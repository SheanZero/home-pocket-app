# Home Pocket V16 Mockup

V16 由 V15 整 App Mockup 升版而来，并于 2026-07-16 与当前 Flutter App 的主页面及统一记账实现重新同步。

## 打开方式

直接在浏览器打开 [index.html](index.html)。Mockup 为单文件交互稿；Material Symbols 图标字体通过 Google Fonts 加载。

## V16 同步内容

- 主页、明细、统计、购物统一使用同一主标题栏规范：顶部 11px、左右 20px、标题栏高度 46px、标题 20/28/700、标题栏与正文间距 13px。
- 主标题在左；右侧操作按钮为 40×40，图标为 24px。主页、明细、统计保留日历与设置，购物页补齐设置入口。
- 主页模式徽标同步为 27px 最小高度、11/15/700 字体与 15px 图标。
- 购物筛选恢复“私有”和“类目”：个人模式显示账簿段控件与二级筛选；家庭模式额外显示“全部 / 个人”范围段控件。
- 私有、类目和账簿筛选可以组合；类目使用购物专用的 L1 多选底部面板，支持清空、取消和应用。
- 家庭范围切换会重置账簿、私有与类目筛选，与 App 状态逻辑一致。
- 购物项新增 / 编辑表单已按当前 `ShoppingItemFormScreen` 重做：空名称、数量 1、日常、公开为新增默认值；字段顺序为名称、数量、用途、类型、分类、参考价格和多行备注，保存移到右上角。
- 公开 / 私有不再由个人或家庭模式推断。新增时可选择，编辑时锁定；“保存后不可更改”紧贴在公开 / 私有控件下，只约束类型，数量和用途始终保持可编辑。名称为空会停留当前页提示，提交中会禁用保存。
- 购物项表单加入可切换的语音提案，手动、聆听、解析、识别回填和权限不可用全部留在同一个 `shopping-form` 页面；约 3 秒停顿自动解析，中央按钮可提前结束，并在回填后复用为重新录音。
- 购物语音只填写一个草稿，不自动保存、不新增确认键，也不推断公开 / 私有；用户检查字段后仍由右上“保存”统一提交。录音期间切回键盘恢复原草稿，识别后切回键盘保留回填结果，旧异步结果通过会话序号失效。
- 语音隐私文案使用“端末内認識を優先”，与当前设备端优先、可按设置降级的实现一致，不再无条件声明“仅在设备内处理”。后续 App 实现应复用语音采集、语言和权限能力，并新增购物专用解析结果，而不是复用交易金额解析器。
- 手工与语音记账收敛为唯一的统一记账页面：金额、类目、日期、商家、用途和备注始终挂载，语音只在原位置替换底部数字键盘。
- 统一记账已选定方案 1「单页连续输入」：下一笔待机、聆听、解析、识别结果和权限不可用共用同一个 336px 语音底板，不再保留独立语音页面或识别结果页面。
- 语音底板右上角的 44×44 键盘入口在全部语音状态中保持同一位置；聆听或解析时切换会取消未完成片段，识别后切换会保留已经填入表单的内容。
- 语音演示覆盖下一笔待机、聆听、解析、识别结果和麦克风不可用；停止说话约 3 秒后自动进入解析，悦己色停止按钮只用于立即完成，不再是必经操作，聆听与解析状态不再显示取消按钮。
- 识别结果直接更新同页上方表单；中央麦克风复用为唯一的“重新录音”入口，底部只保留一枚“记录”，同时完成结果确认与保存。
- 手工与语音底板复用同一个连续记账控件。语音连续保存后留在语音底板的“下一笔待机”状态，不自动开启麦克风；用户点按中央麦克风后才开始下一笔。
- 悦己色只保留给语音状态反馈：状态点、波形和中央录音/停止/重录键。语音入口恢复中性卡片并仅用主绿色点缀图标，键盘入口恢复中性色，最终记录使用标准主绿色；来源标记和连续记账也回到普通主色体系，未增加新的颜色。
- 语音填充的字段显示来源标记；低置信度类目要求先选择有效类目。金额在手动与语音状态中始终使用同一份数据。
- 自动结束只触发解析，不会自动保存；识别结果不再增加独立确认步骤，“记录”同时完成结果确认与保存。
- 编辑交易已从旧版居中金额与静态列表重做为统一记账同源结构：52px 居中标题栏、横向金额与固定币种标记、分类 / 日期 / 商家详情卡、用途卡、悦己满足度和独立备注卡全部复用 `entry-*` 组件语言。
- 编辑态只保留真实差异：左侧返回、右上唯一删除入口、金额底部键盘和底部固定绿色保存；不出现语音入口、常驻数字键盘、连续记账或页尾重复删除。商家和备注可直接编辑，日常 / 悦己可切换，删除先确认，保存与删除都返回原来源页。
- 编辑交易 Inspector 增加 JPY 日常、JPY 悦己、USD 外币和保存中四种演示。外币头部显示原币金额，独立换算卡展示汇率、只读日元结果和汇率日期，币种本身不可在编辑态改变。

- A1：个人 · 浅色
- A2：家庭 · 浅色
- A3：个人 · 深色
- 视口快捷宽度：375 / 390 / 430
- 购物项表单状态：手动 / 聆听 / 解析 / 识别回填 / 编辑 / 麦克风不可用
- 统一记账方案：方案 1「单页连续输入」；唯一 `entry` 页面 + 固定 336px 语音底板；端点模式：3 秒停顿自动解析 + 悦己色按钮立即完成；演示状态：手动 / 聆听 / 识别结果 / 下一笔待机 / 麦克风不可用
- 编辑交易状态：JPY 日常 / JPY 悦己（5 档满足度）/ USD 外币联动 / 保存中 / 删除确认；视觉组件与统一记账同源，交互状态独立于新增账目草稿
